[PROJECT_OVERVIEW]

The Autonomous Ground Control Station V2 (GCS V2) represents a new generation of field-deployable command centers. Designed to facilitate complex multi-platform operations involving UAVs and UGVs, it integrates high-speed, military-grade data links, automated antenna tracking systems, and an optimized operator interface.

Key Features: - Aircraft Control Stick - Radiomaster TX12 Controller - Control Sticks, Toggle Switches, and Sliders - 6 I/O Ports (RJ45, HDMI, USB-A, USB-C) - Dual Monitors - Active Cooling System
